This repository contains the core framework of the QuithorDDD-MVC architecture, which can be built into a Maven dependency to quickly implement Spring projects. It also includes subsequent AI plugin and scheduled task plugin, which are DLCs implemented using this framework.

## Project Introduction

QuithorDDD-MVC-core is an intermediate framework guided by Domain-Driven Design (DDD) based on MVC architecture. Built as a Maven dependency, it enables rapid development of Spring projects. The framework provides extensible plugin support, including AI plugins and scheduled task plugins as DLC (Downloadable Content) modules.

## Features

- **DDD-Guided MVC Architecture**: Combines DDD principles with traditional MVC pattern to provide a clean and maintainable architecture
- **Quick Spring Integration**: Built as a Maven dependency for rapid integration into Spring projects
- **AI Plugin Support**: Extensible AI functionality through dedicated plugins
- **Scheduled Task Plugin**: Built-in support for scheduled tasks via plugin architecture
- **DLC Expansion**: Modular design allowing additional features through DLC-style plugins

## Usage

This framework can be built into a Maven dependency that allows for quick implementation of Spring projects. Include it in your project to leverage the DDD-guided MVC architecture and available plugins.

## Project Structure

- **Core Framework**: The main DDD-guided MVC framework
- **AI Plugin**: AI functionality extension
- **Scheduled Task Plugin**: Task scheduling extension
- **DLC Modules**: Additional feature extensions using the framework
