# QuithorDDD-MVC-core

中文版 | [English Version](README_en.md)

## 项目简介

这是一个基于DDD（领域驱动设计）指导的MVC架构的中间框架。该框架旨在帮助开发者快速构建Spring项目，提供了一套完整的架构解决方案。

## 核心特性

- **DDD指导的MVC架构**：结合领域驱动设计思想，提供清晰的分层架构
- **快速集成Spring项目**：构建的Maven依赖项可快速实现Spring项目开发
- **AI插件支持**：提供后续的AI功能插件扩展
- **定时任务插件**：内置定时任务插件，便于任务调度管理
- **DLC扩展**：框架的附加功能模块，提供更丰富的开发能力

## 使用方式

将此仓库构建的Maven依赖项添加到您的项目中，即可快速实现Spring项目的开发。

## 项目结构

- 核心框架：提供基础的MVC架构支持

## 画饼部分

- AI插件：智能功能扩展模块
- 定时任务插件：任务调度功能模块
